# Tockish Protocol

The open protocol for representing, linking and verifying collaborative contributions.

## Mission

Every great thing started as tockish.

The Tockish Protocol defines a common representation for collaborative contributions, enabling projects to preserve their history, relationships and decision-making process.

> Git remembers code. Tockish remembers collaboration.
